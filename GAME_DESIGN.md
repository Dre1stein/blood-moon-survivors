# 🧛 Blood Moon Survivors — 吸血鬼Like游戏设计文档

## 🎮 核心概念

**Blood Moon Survivors** — 在血月之夜，你扮演一个觉醒的吸血鬼猎人，在无尽被诅咒的地图中生存。

### 核心玩法循环
移动 + 自动攻击 → 击杀怪物 → 获取经验/掉落 → 升级选技能 → 合成进化 → 活过30分钟

---

## 🗺️ 技术方案

- **框架**: Phaser 3 (TypeScript) — 2D Canvas，成熟的游戏框架
- **构建**: Vite
- **目标**: 浏览器游戏，单文件部署

### 为什么选 Phaser 3
- 你之前 xjjy 项目用过 Phaser，有经验
- 内置物理引擎、碰撞检测、粒子系统
- 丰富的武器/特效支持
- TypeScript 类型完善

---

## 🎯 局内系统 (In-Run)

### 1. 角色属性
- HP / Max HP
- 移动速度
- 拾取范围
- 护甲（减伤）
- 暴击率 / 暴击伤害
- 冷却缩减

### 2. 武器系统（最多6把）

| 武器 | 类型 | 描述 | 升级效果 |
|------|------|------|----------|
| 银弹手枪 | 定向投射 | 向最近敌人射子弹 | +1子弹/伤害 |
| 圣水飞瓶 | 范围投射 | 抛物线投掷，落地爆炸 | +范围/伤害 |
| 十字架 | 旋转环绕 | 围绕角色旋转 | +数量/速度 |
| 神圣光环 | 持续AOE | 角色周围持续伤害 | +范围/伤害 |
| 血刃匕首 | 定向穿透 | 高速直线穿透 | +伤害/穿透次数 |
| 雷霆之锤 | 锁定打击 | 随机锁定敌人坠落 | +打击次数/伤害 |
| 寒冰结界 | 地面AOE | 周期性释放冰冻波 | +范围/冻结时间 |
| 死亡凝视 | 射线 | 向最近敌人发射激光 | +伤害/穿透 |

### 3. 被动技能（最多6个）
- 移速提升
- HP 回复（每秒）
- 伤害加成
- 暴击率提升
- 拾取范围增大
- 护甲提升
- 经验获取+20%
- 冷却缩减
- 生命窃取（造成伤害的5%回血）

### 4. 武器合成（进化）
同类型武器+特定被动 = 进化武器
- 银弹手枪 + 暴击率 = **银色审判**（子弹追踪+爆炸）
- 十字架 + 移速 = **天使之环**（更大范围+弹射）
- 圣水飞瓶 + 拾取范围 = **圣火洗礼**（持续燃烧地面）
- 神圣光环 + HP回复 = **生命之泉**（光环内回复HP）
- 血刃匕首 + 生命窃取 = **血月斩**（吸血+更大刀气）
- 雷霆之锤 + 冷却缩减 = **雷神之怒**（连续闪电链）

### 5. 怪物设计

**普通怪（波次生成）：**
- 🧟 丧尸 — 慢速，低HP，近战
- 🦇 蝙蝠 — 快速，极低HP，小体型
- 👻 幽灵 — 中速，会穿墙
- 🐺 狼人 — 中速，中HP，冲刺攻击
- 🧛 小吸血鬼 — 会远程攻击（吐血弹）

**精英怪（每5分钟出现）：**
- 💀 死灵骑士 — 大型，高HP，召唤小怪
- 🐉 暗影龙 — 飞行，范围喷火
- 👁️ 邪眼 — 激光扫描，高伤害

**BOSS（每10分钟）：**
- 第10分钟：**血伯爵** — 召唤蝙蝠群，血遁术
- 第20分钟：**暗月女巫** — 毒雾+诅咒debuff
- 第30分钟：**血月魔王** — 最终BOSS，全屏攻击+分身

### 6. 地图系统

**初始地图：被诅咒的村庄**
- 草地 + 破败建筑 + 墓地
- 随机生成障碍物（不可穿越）
- 地图随时间扩大（视野限制→雾战）

**解锁地图（局外养成解锁）：**
- 🌲 暗影森林 — 树木遮挡，更多远程怪
- 🏰 废弃城堡 — 室内迷宫，精英密集
- ⛪ 堕落教堂 — Boss专属地图，高难度
- 🌙 血月荒野 — 无尽模式，极限挑战

## 🏠 局外养成系统 (Meta Progression)

### 1. 角色系统
- 初始角色：**猎人**
- 解锁角色（金币购买）：
  - 🧙 **法师** — 初始冷却-15%，伤害+10%
  - 🏹 **游侠** — 初始移速+20%，拾取+30%
  - 🛡️ **圣骑士** — 初始HP+30%，护甲+10
  - 🐺 **狼人** — 近战伤害+25%，但远程-20%

### 2. 技能树（局外永久升级）
用局内收集的**血石**升级，每项3级

**生存分支：**
- HP上限 +10/20/30%
- 回复速率 +0.5/1/2 每秒
- 初始无敌时间 +1/2/3 秒

**攻击分支：**
- 全伤害 +5/10/15%
- 暴击率 +3/6/10%
- 暴击伤害 +25/50/75%

**效率分支：**
- 经验获取 +10/20/30%
- 拾取范围 +15/30/50%
- 移动速度 +5/10/15%

### 3. 装备系统
局内掉落装备，局外永久持有，可装备3件

| 装备槽 | 示例 | 效果 |
|--------|------|------|
| 武器 | 吸血鬼猎人之弩 | 伤害+15% |
| 护甲 | 月光斗篷 | 减伤+10% |
| 饰品 | 血红宝石 | 生命窃取+3% |

装备稀有度：白 → 绿 → 蓝 → 紫 → 🟡 橙（传奇）

### 4. 符文槽系统
- 3个符文槽位
- 符文类型：🔥火（伤害）、❄️冰（控制）、⚡雷（速度）、💚生（回复）、🌑暗（暴击）
- 符文等级 1-5，可合成升级
- 效果：全局buff，叠加基础属性

### 5. 成就系统
- "初次觉醒" — 完成第一局
- "血月猎人" — 击杀10000怪
- "生存大师" — 存活30分钟
- "进化之路" — 触发一次武器进化
- 等等...
